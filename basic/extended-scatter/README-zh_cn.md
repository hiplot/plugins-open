# 拓展散点图

- 简介

  在散点图的基础上拓展边缘图像

- 特殊参数

  add_rug: 是否在边缘添加rug
  
  rug_alpha: rug的透明度
  
  rug_size: rug的大小
  
  rug_color: rug的颜色
  
  marginal_type: 边缘图的类型，可选项为：histogram, boxplot, violinplot, qqplot, ppplot, densigram
  
  marginal_fill: 边缘图的填充颜色
  
  marginal_color: 边缘图的边框颜色
  
  marginal_size: 边缘图的线宽
  
  marginal_bins: 边缘图的柱形数量
